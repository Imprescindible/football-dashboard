# Premier League Dashboard

Футбольный дашборд АПЛ сезона 2023/24, построенный на Vite + React + Redux + TypeScript.

## Технологии

- **React 18** + **TypeScript**
- **Redux Toolkit** — управление состоянием, кэширование данных
- **React Router v6** — навигация, динамические маршруты
- **Framer Motion** — анимации переходов между страницами
- **football-data.org API** — данные матчей

## Функционал

### Страница матчей

- Все 380 матчей сезона
- Фильтрация по туру, дате и названию команды
- Фильтры сохраняются в URL
- Гербы команд

### Таблица

- Сортировка по любой колонке
- Гербы команд
- Сравнение двух команд в модальном окне
- Топ-5 команд по забитым и пропущенным голам

### Страница матча

- Счёт и статус матча
- Переход на страницу любой команды

### Страница команды

- Герб, название
- График формы последних 5 матчей
- Статистика сезона — победы, ничьи, поражения, голы
- Фильтр матчей — все, дома, в гостях
- Полная история матчей с результатами

## Запуск

```bash
# Установить зависимости
npm install

# Создать .env файл
echo "VITE_FOOTBALL_API_KEY=ваш_ключ" > .env

# Запустить
npm run dev
```

## Получить API ключ

Бесплатный ключ на [football-data.org](https://www.football-data.org/client/register)

## Структура проекта

```
src/
  api/          # Запросы к API
  components/   # Переиспользуемые компоненты
  hooks/        # Кастомные хуки
  pages/        # Страницы приложения
  store/        # Redux слайсы
```
